About Simple Artifact Collection

Simple Artifact Collection is a secure and scalable platform for uploading, managing, and organizing artifacts. It leverages Azure services for storage and authentication, ensuring a modern and reliable experience.

Features

  • File Upload Portal: Users can upload files to specific cases using a secret key.
  • Admin Portal: Admins can create cases, view case details, rotate secrets, and manage files in blob containers.
  • Secure File Storage: Files are compressed and password-protected before being uploaded to Azure Blob Storage.
  • Azure Integration:
    • Azure Blob Storage for secure file storage.
    • Azure SQL Database for case metadata storage.
    • Microsoft Entra ID (Azure AD) for authentication.
  • Dynamic UI: A responsive and interactive user interface built with modern web technologies.
  • Generate Upload Links: Admins can generate pre-filled upload links for specific cases.

How It Works

Users upload files to specific cases using a secret key. Admins can create and manage cases, which are mapped to Azure Blob Storage containers. Files are compressed and secured with a password before being stored. The system ensures secure file storage and metadata management using Azure services.
