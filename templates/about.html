About File Management System

The File Management System is a secure and scalable platform for uploading, managing, and organizing files. It leverages Azure services for storage and authentication, ensuring a modern and reliable experience.

Features

  • File Upload Portal: Users can upload files to specific cases using a secret key.
  • Admin Portal: Admins can create cases, view case details, and manage files in blob containers.
  • Azure Integration:
    • Azure Blob Storage for secure file storage.
    • Azure SQL Database for case metadata storage.
    • Managed Identity for secure authentication to Azure services.
  • Dynamic UI: A responsive and interactive user interface built with modern web technologies.

Technologies Used

  • Flask (Python) for the backend.
  • Azure Blob Storage and Azure SQL Database for storage and metadata management.
  • Microsoft Entra ID (Azure AD) for authentication.
  • Tailwind CSS and Font Awesome for a modern and responsive UI.

How It Works

Users upload files to specific cases using a secret key. Admins can create and manage cases, which are mapped to Azure Blob Storage containers. The system ensures secure file storage and metadata management using Azure services.
